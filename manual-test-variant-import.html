Manual Test: Import Flow CSS Injection for Variants

Test Purpose:

This test verifies that imported components with variant selectors properly inject their template CSS, allowing variant updates to have visual effects.

Expected Behavior:

  • ✅ Imported button component should display with proper styling
  • ✅ Variant changes (primary/secondary/danger) should be visually apparent
  • ✅ Size changes (small/medium/large) should be visually apparent
  • ✅ CSS should be injected into the page automatically
